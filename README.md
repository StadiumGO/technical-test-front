# Test technique StadiumGO

### Getting Started

1 : Clone the repository and cd into it

2 : `npm install`

3 : `npm run dev`

### Verifications

Prettier : `npm run prettier` or `npm run prettier:check`

### Start Dev !

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
